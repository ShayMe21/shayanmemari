Orion Network Tool

Details
  • Category: Web Application
  • Date: 2017
  • Languages: PHP, Java, MySQL
  • Technologies: Symfony, Doctrine, Composer, Kendo UI, LDAP
Description
A network management and inventory tool for Vodafone used as an internal tool with real time syncing of network equipment between multiple servers. LDAP was integrated for authentication into the application using OpenLDAP library. All modules later added were developed by adhering to existing MVC, DDD and OO principles as well PHPUnit testing to ensure code reusuability and standardised code.
