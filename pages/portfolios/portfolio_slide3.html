A static site was created for Rock Steel Group Pty Ltd from initial concept to deployment on AWS. A responsive Single-page application was created using Angular 4 with a full screen modal gallery and contact form with custom validation using reactive forms and validators module. Automated deployment with AWS CLI and webpack on AWS S3 Bucket.
